## 功能 1️⃣：优雅隐藏顶栏和侧边栏

> 🥡 上划显示，下滑隐藏

> 😶‍🌫️ 半透明的边栏，用了才知道有多棒

## 功能 2️⃣：简化复杂的划线菜单

> 🍵 简简单单划线，简简单单读书

> 🤠️ 还找不到？到`孔夫子`或`多抓鱼`淘本便宜的二手书~
