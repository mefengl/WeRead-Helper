## 功能 1️⃣：优雅隐藏顶栏和侧边栏

> 🥡 上划显示，下滑隐藏
>
> 😶‍🌫️ 半透明的边栏，用了才知道有多棒

## 功能 2️⃣：简化复杂的划线菜单

> 📌 简简单单的菜单，简简单单的划线

## 功能 3️⃣：一键搜豆瓣、得到电子书，还可在孔夫子、多抓鱼买二手

> 🕵️‍♀️ `微信读书`查无此书？到`豆瓣阅读`或`得到阅读`试试吧
>
> 🤠️ 还找不到？到`孔夫子`或`多抓鱼`淘本便宜的二手书~
