# 📘微信读书阅读助手

读书人用的脚本。

## 功能 1️⃣：自动隐藏顶栏和侧边
> 🥡 上划显示，下滑隐藏

## 功能 2️⃣：半透明顶栏和侧边
> 😶‍🌫️ 小小的改动，用了才知道有多棒。

## 功能 3️⃣：宽度保持

> ✅ 在窄屏下阅读时，优先缩小两边空隙，而不是文字栏。

## 声明 👀

> 📝 本项目[WeRead-Helper](https://github.com/mefengl/WeRead-Helper)开源，欢迎读书爱好者在GitHub提issue，优化代码。

> 🎈 本脚本参考了xvusrmqj的“微信读书加宽度”和Lossj的“微信读书沉浸式阅读”的功能。
